# AWS SAA-C03 模擬試験アプリ

AWS Solutions Architect Associate（SAA-C03）の本番試験に近い形式で学習できる、シングルHTMLファイルの模擬試験アプリです。

## デモ

`aws-saa-exam.html` をブラウザで開くだけで動作します（サーバー不要）。

## 機能

| 機能 | 詳細 |
|---|---|
| 問題数の選択 | 5〜65問をスライダーで選択（5問刻み） |
| 自動タイマー | 問題数 × 2分（65問 = 130分） |
| 問題形式 | 単一選択・複数選択の混在 |
| フラグ機能 | 後で見直したい問題に🚩を付けて保留 |
| ナビゲーター | 全問の回答状況を色分けで一覧表示 |
| 見直し画面 | 提出前に回答済み／フラグ／未回答を確認 |
| スコア算出 | スケールドスコア（100〜1000点、合格720点） |
| ドメイン別結果 | 4ドメインの正答率をグラフで表示 |
| 解答レビュー | 全問の正誤・解説を提出後に確認 |

## 出題範囲

SAA-C03の4ドメインに対応した75問バンクから、毎回ランダムで出題されます。

| ドメイン | 内容 | 本番の比率 |
|---|---|---|
| Domain 1 | セキュアなアーキテクチャの設計 | 30% |
| Domain 2 | 弾力性の高いアーキテクチャの設計 | 26% |
| Domain 3 | 高パフォーマンスなアーキテクチャの設計 | 24% |
| Domain 4 | コスト最適化されたアーキテクチャの設計 | 20% |

## 使い方

1. `aws-saa-exam.html` をブラウザで開く
2. スライダーで問題数を選択
3. 「試験を開始する」をクリック
4. 回答・フラグを使いながら全問を解く
5. 「見直し」→「提出する」でスコアを確認
6. 「解答を確認する」で全問の解説を読む

## 技術スタック

- HTML / CSS / JavaScript（バニラ、依存ライブラリなし）
- [Tailwind CSS](https://tailwindcss.com/)（CDN経由）
- ブラウザのみで動作（Node.js・サーバー不要）
